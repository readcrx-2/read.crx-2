read.crx 2

read.crx 2はGoogle Chrome/Microsoft Edge/Mozilla Firefox extensionとして作られた2chブラウザです。 2ch.net、2ch.sc、open2ch.net及びその互換BBS、まちBBS、したらばの閲覧に対応しています。

インストール手順(Chrome/Edge)

Windows

Windowsでは、制限があるため、下記のいずれかの方法で、制限を回避してインストールする必要があります。

デベロッパーモードを使用する方法

  1. 上記ダウンロードのリンクより、crxファイルをダウンロードします。
  2. ダウンロードしたcrxファイルの拡張子を「zip」に変更し展開します。（標準のアーカイバでは正常に展開できないため、7zipなどを使用して下さい）
  3. 必要があれば、展開されたディレクトリを邪魔にならない場所に移動します。
  4. 拡張機能の管理画面で、「デベロッパーモード」にチェックを入れます。
  5. 拡張機能の管理画面で、「パッケージ化されてない拡張機能を読み込む…」ボタンをクリックし、3のディレクトリを選択します。

レジストリーエディタを使用する方法

  1. readcrx2.reg（右クリックからダウンロードして下さい。）をダウンロードします。
  2. ダウンロードしたregファイルを実行します。
  3. 上記Chrome/Edge版のダウンロードのリンクを左クリックし、インストールします。

Mac OS X

Mac OS Xでも、Windowsと同様に制限がかかり始めたため、デベロッパーモードを使用する方法もしくは、下記の方法で制限を回避してインストールする必要があります。

  1. readcrx2.mobileconfig（右クリックからダウンロードして下さい。）をダウンロードします。
  2. ダウンロードしたmobileconfigファイルを開きます。 管理者権限を持っているユーザである必要があります。また、実行時にログインパスワードを求められます。
  3. 上記Chrome/Edge版のダウンロードのリンクを左クリックし、インストールします。

Linux

  1. 上記ダウンロードのリンクより、crxファイルをダウンロードします。
  2. 拡張機能の管理画面へ、ダウンロードしたcrxファイルをドラッグ&ドロップします。

インストール手順(Firefox)

  1. 上記ダウンロードのリンクを左クリックします。
  2. 左上に「このサイトからはFirefoxにソフトウェアをインストールできない設定になっています。」と出てきたら、「許可する」をクリックします。
  3. さらに左上に「read.crx 2を追加しますか？あなたの許可が必要です:」と出てくるので、「追加」をクリックします。

商用利用時の注意
